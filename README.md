# Playback Store (Учебный проект по ASP.NET Core Razor Pages)


Учебный проект “Playback Store” реализован на ASP.NET Core с использованием Razor Pages и Entity Framework Core.  


## Цели проекта

- Реализовать CRUD-функционал (создание, просмотр, редактирование, удаление) для каждой таблицы.
- Использовать базу данных.
- Обеспечить проверку вводимых значений (валидацию) через DataAnnotations.
- Сделать удобный интерфейс с единым меню.
- Добавить XML-документацию для моделей и PageModel.


## Используемые технологии

- ASP.NET Core Razor Pages
- Entity Framework Core
- PostgreSQL (через Npgsql)
- C#
- Visual Studio 2022 / VSCode


## Модели и связи

- Genre – музыкальный жанр (обязательное имя, описание необязательное)  
- Band – музыкальная группа (обязательное имя, количество участников, связь с жанром)  
- Playback – музыкальная дорожка (обязательное название, длительность, цена, связь с группой)  

Связи:  
- Band → Genre (каждая группа имеет один жанр)  
- Playback → Band (каждый плейбек принадлежит одной группе)


## Валидация

- Валидация реализована через DataAnnotations:
  - `[Required]` – обязательные поля
  - `[StringLength]` – ограничение длины текста
  - `[Range]` – числовые диапазоны
  - `[Display(Name = "...")]` – подписи полей на русском
- Проверка на сервере: `ModelState.IsValid`  
- Клиентская валидация подключена через `_ValidationScriptsPartial.cshtml`  


## Интерфейс

- Единый шаблон `_Layout.cshtml` для всех страниц  
- Навигационное меню: Жанры | Группы | Плейбеки  
- Таблицы с кнопками действий: Создать, Редактировать, Детали, Удалить
